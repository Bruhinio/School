# SchoolLMS — Sistema de Gestión Escolar

Plataforma web para gestión escolar con roles de Directora, Docentes y Alumnos.

## 🚀 Setup

### 1. Instalar dependencias
```bash
npm install
```

### 2. Configurar Firebase
- Copia `.env.example` → `.env.local`
- Llena tus credenciales de Firebase Console

### 3. Correr en local
```bash
npm run dev
```

### 4. Build para producción
```bash
npm run build
```

## 🔧 Stack
- React 18 + Vite
- Firebase (Firestore + Auth)
- PWA (vite-plugin-pwa)
- Deploy en Vercel
